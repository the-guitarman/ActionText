- (Object) format_interpunctuation(str)

A Bunch of transformations to style text in a non-ugly text format. For Example take this String:

"A test StringAnother Sentence!!!The End - Really, the end."

after calling this method, you will get the following string:

"A test String. Another Sentence! The End. Really, the end."

def format_interpunctuation str
	new_str = str.dup
	new_str.gsub!(" - ", ". ")
	new_str.gsub!(/([a-z|äöüß])([A-Z|ÄÖÜ])/) { "#{$1}. #{$2}" }
	new_str.gsub!(/\!+/,"!")
	new_str.gsub!(/\?+/,"?")
	new_str.gsub!(/([\!|\?|\.])([\w|äöüÄÖÜß])/) { "#{$1} #{$2}"}
	new_str
end

- (Object) format_remove_non_letters(str)

This simple formatter deletes every character from the string, which is usually not used in texts. Also it purges ?!. and prepares the string to get split into single words. For a far better experience, use the format_interpunctuation method first!

def format_remove_non_letters str
	new_str = str.dup
	new_str.gsub!(/[^\w|öäüÖÄÜß|\ |\'|\"']/,'')
	new_str
end

def split_to_words str, flags={}
	format = flags[:format] || flags['format'] || false
	str = format_interpunctuation(str) if format
	string_without_special_signs = format_remove_non_letters(str)
	string_without_special_signs.split(" ")
end
